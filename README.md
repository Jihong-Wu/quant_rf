# Algorithmic Trading Strategy Project

## Objective

本项目实现了一个端到端的机器学习量化交易研究流程。通过历史市场数据提取多维度因子，训练随机森林分类模型预测短周期价格方向，并据此生成系统化交易信号，用于量化策略研究与展示。

---

## Features

* 基于 10 年以上历史市场数据进行建模
* 自动完成市场数据预处理与时间序列结构化
* 构建趋势、动量、波动率、成交量和相对位置等多类因子
* 使用 Random Forest Classifier 进行方向预测
* 输出上涨 / 中性 / 下跌三分类结果
* 支持样本外评估，用于验证模型泛化能力

---


## Data Source

本项目使用历史市场数据作为研究输入。原始数据经过整理后被转换为固定时间粒度的序列，用于后续特征工程和模型训练。

---

## Program Structure

项目中的各个程序分别承担不同功能，共同组成完整的量化研究流程。主要模块包括：

* **main.py**
  主程序入口。负责串联整个流程，包括数据读取、因子构建、标签生成、模型训练、预测输出与结果评估。

* **data.py**  
  数据处理模块。负责读取原始市场数据，并完成基础清洗、整理、格式转换和时间序列聚合，为后续因子工程和模型训练提供标准化输入数据。


* **indicators.py**
  因子工程模块。用于计算各类技术指标和统计特征，例如趋势、动量、波动率、成交量以及价格相对位置等因子，为模型提供输入特征。

* **Random_Forests_model.py**
  机器学习建模模块。用于训练随机森林分类模型，并根据特征数据预测未来价格方向。

如果项目后续扩展更多脚本，也可以进一步拆分为数据预处理、回测分析、风险控制和可视化等模块，以增强代码结构清晰度和可维护性。

---

## Methodology

1. **Data Processing**：对历史市场数据进行清洗、整理与聚合。
2. **Feature Engineering**：构建价格、成交量和波动相关因子。
3. **Label Construction**：依据未来价格变化生成分类标签。
4. **Machine Learning**：使用随机森林模型学习因子与未来方向之间的关系。
5. **Signal Generation**：将模型预测结果转化为交易信号。
6. **Evaluation**：对样本外结果进行表现分析。
